# Personal Portfolio Website

A responsive personal portfolio website built using HTML, Bootstrap, and CSS.  
This portfolio showcases my projects, skills, and information as an aspiring Full Stack Developer.

## 🚀 Features

- Responsive design using Bootstrap
- Clean and modern user interface
- About Me section
- Skills section
- Projects showcase
- Contact section
- Smooth navigation
- Mobile-friendly layout using media queries

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## 📂 Sections Included

- Home
- About
- Skills
- Projects
- Contact

## 🎯 Purpose of This Project

The purpose of this portfolio is to:
- Showcase my frontend development skills
- Display my projects
- Practice responsive web design
- Build a professional online presence
  
This portfolio contains:
- Hero section with introduction
- Project cards with GitHub links
- Responsive layout for different screen sizes
